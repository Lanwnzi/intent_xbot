## 🎯 关键文件详解

### ⭐⭐ backend/app.py
- **路径**：`backend/app.py`
- **职责**：FastAPI 应用入口，注册所有路由，初始化 AgentManager、Checkpointer、MemoryIndexer
- **核心方法/类签名**：
  ```python
  # backend/app.py:21-30
  @asynccontextmanager
  async def lifespan(_: FastAPI):
      await init_checkpointer_async()
      refresh_snapshot(settings.backend_dir)
      agent_manager.initialize(settings.backend_dir)
      memory_indexer.configure(settings.backend_dir)
      memory_indexer.rebuild_index()
  ```
- **修改频率**：低

## 🔄 代码流向示意

### 典型对话请求流程：

```
1. 用户发送消息
   → Frontend (Next.js)
   
2. HTTP POST /api/chat
   → backend/api/chat.py:chat()

3. 加载会话历史
   → backend/service/session_manager.py:load_session_record()

4. 构建 Agent
   → backend/graph/agent_factory.py:create_agent_from_config()
   
5. 组装 System Prompt
   → backend/service/prompt_builder.py:build_system_prompt()

6. (v2 always 模式) 检索记忆并注入
   → backend/memory_module_v2/integrations/middleware.py:build_memory_context()

7. Guardian 前置审查
   → backend/graph/guardian.py:GuardianMiddleware.before_agent()

8. Agent 执行 (LLM + Tools)
   → LangChain create_agent 内部

9. SSE 流式响应
   → backend/graph/agent.py:astream() yield token/tool_end/done

10. 保存消息到 JSON
    → backend/service/session_manager.py:save_message()

11. (v2 模式) 后台蒸馏
    → backend/memory_module_v2/service/api.py:distill_session()
```

## 🎓 新手入门指南

### 如何扩展项目

**添加新工具**：
1. 在 `backend/tools/` 创建新工具类，继承 `BaseTool`
2. 在 `backend/tools/__init__.py:get_all_tools()` 中注册
3. 工具自动被 Agent 感知

**添加新技能**：
1. 在 `backend/skills/` 创建目录
2. 添加 `SKILL.md` 定义技能
3. 下次对话自动扫描并加载

**修改 System Prompt**：
1. 编辑 `backend/workspace/*.md` 中的对应文件
2. 无需重启，下轮请求自动生效
